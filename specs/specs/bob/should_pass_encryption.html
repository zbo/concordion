Encryption

this library should provide encryption and decryption for user password

Example

When user World logs in, the greeting will be: Hello, World!

When user input password as P@ssword actual password saved to database should be: XvpjmDGOHel2r+gYMoV7kQ==

When system retrieve encrypted password as XvpjmDGOHel2r+gYMoV7kQ== decrypted password should be: P@ssword
